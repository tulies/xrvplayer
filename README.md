# XRV PLAYER

这是一个h5-video播放器,目前还在开发中....

# 如何查看演示

- 第一步：进入example目录
- 第二步：执行 `http-server` 。你需要提前安装好 `http-server` ；若你还没安装，请先安装：`npm install -g http-server` 。


## 为什么要造轮子

造轮子只是为了适应自己的业务，主要是为了开发多视角、自由视角、VR等播放器。

这些业务场景后续将会以 `plugin` 形式插入。
